# promter

### Назначение
При работе с chatGPT возникает необходимость передать нейросети контекст разработки (листинги кода) в окно чата.

Для избежания ручного копипастинга был создан Promter. Он осуществляет сбор и компоновку в один промт листинга всех модулей из папки с проектом.

### Типы сканируемых файлов указаны в переменной в модуле src/file_extensions.py
file_extensions = ['.txt', '.md', '.log', '.csv', '.py', '.cfg', '.conf', '.json', '.yml', '.yaml']

### Зависимости
установка зависимости: python -m pip install -r requirements.txt

### Использование
Скрипт обходит директорию проекта (включая поддиректории) и собирает содержимое файлов с раширениями, указанными в `file_extensions` в один текстовый файл.
После обработки содержимое всех файлов записывается в 'output.txt' в директории promter, а также копируется в буфер обмена.

Скрипт принимает следующие необязательные параметры:

    --repo_path - Путь к папке с репозиторием, по умолчанию '.'
    --output - Имя файла для сохранения содержимого, по умлочанию 'output.txt'
    --no_copy - Не копировать в буфер

Пример запуска:
`python3 <path_to_promter_dir>/promter.py --repo_path /path/to/my/repository --output /path/to/output/output.txt`
